# 🛍️ RetailPulse – AI-Powered Retail Decision Platform

An end-to-end Business Intelligence application built using **Python**, **Streamlit**, **Plotly**, and **Prophet** to transform retail transaction data into actionable business insights.

RetailPulse helps business stakeholders monitor sales performance, understand customer behavior, optimize inventory decisions, and forecast future revenue through interactive dashboards.

## 📌 Project Overview

Retail businesses generate large volumes of transactional data every day. Extracting meaningful insights from this data is essential for improving customer retention, inventory planning, product strategy, and future revenue forecasting.

RetailPulse addresses these challenges by combining exploratory data analysis, business dashboards, customer segmentation, inventory analytics, and time-series forecasting into a single interactive analytics platform.

# 📂 Dataset

**Online Retail II Dataset (UCI Machine Learning Repository)**

The dataset contains transactional records from a UK-based online retailer between **2009 and 2011**.

Features include:

- Invoice Number
- Product Description
- Quantity
- Unit Price
- Customer ID
- Country
- Invoice Date

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/RetailPulse.git
```

Navigate to the project

```bash
cd RetailPulse
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
